# Window Image References

Put window reference images in this folder.

Recommended file naming:

```text
country-window-type-source-or-keyword.jpg
```

Examples:

```text
japan-shoji-soft-light.jpg
france-casement-balcony.jpg
china-lattice-garden.jpg
morocco-mashrabiya-pattern.jpg
italy-shutters-sunlight.jpg
industrial-sash-window.jpg
```

When adding an image, also record the source link and design notes in:

```text
../../window-interaction-system.md
```

Do not use images in the public website unless their license is clear. For now these images are for research only.
